# Untitled-Card-Game

Untitled Card Game is a small game prototype I made in about a week's time, there's a tutorial included in the game.

## How to run it

There's three options:
1. Download the zip file in releases, unzip it and run UntitledCardGame.exe inside.

2. Install Löve2D from https://love2d.org, and run the file named UntitledCardGame.love located in the root of this repo.

3. With Löve2D installed, it is also possible to run the source code itself, more information on how to do this can be found on https://love2d.org/wiki/Getting_Started
